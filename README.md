# WT Assignment 04 – Laravel Task Management System

A modern Task Management Web Application built with the Laravel framework following the MVC architecture. This project allows users to securely manage daily tasks with authentication, authorization, dashboard statistics, and full CRUD functionality.

---

## Features

- User Registration & Login
- Secure Authentication using Laravel Breeze
- Create, Edit, Delete Tasks
- Mark Tasks as Completed/Pending
- Dashboard with Task Statistics
- User Profile Management
- Authorization Policies
- Pagination Support
- Responsive UI with Blade & Tailwind CSS
- SQLite Database Integration

---

## Technologies Used

- PHP
- Laravel Framework
- Blade Templating Engine
- SQLite Database
- Eloquent ORM
- Laravel Breeze
- Tailwind CSS
- Vite
